# TS-test

Ts + Vue2.x + vue-router + vuex + axios

cssReset: normalize.css

UI: vant

插件：

qrcode: 二维码生成

html2cavas: canvas工具

immutable: 不可变数据处理

antv/f2: 阿里图表组件

dayjs: 替代momentjs的时间处理库

clipboard2: 复制文本到剪贴板
